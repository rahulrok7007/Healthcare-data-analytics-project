# Healthcare Data Analytics Project

## Project Overview
This project analyzes hospital healthcare data to understand patient visits, doctor workload, diagnosis trends, and hospital revenue using SQL and Excel dashboard.

## Tools Used
- SQL
- Excel Dashboard
- Data Analysis
- Data Visualization

## Dataset
The dataset contains hospital patient records including visits, doctors, lab tests, treatment cost, and diagnosis details.

## Key KPIs Analyzed
1. Total Patients
2. Total Doctors
3. Total Visits
4. Average Age of Patients
5. Top 5 Diagnosed Conditions
6. Follow-Up Rate
7. Treatment Cost Per Visit
8. Total Lab Tests Conducted
9. Percentage of Abnormal Lab Results
10. Doctor Workload (Average Patients per Doctor)
11. Total Revenue

## SQL Analysis
SQL queries were used to:
- Calculate patient visits
- Analyze doctor workload
- Identify top diagnoses
- Calculate hospital revenue

## Excel Dashboard
An interactive dashboard was created to visualize KPIs such as:
- Patient distribution
- Diagnosis trends
- Revenue analysis
- Doctor workload

## Project Files
- SQL Queries: sql_queries.sql
- Dataset: healthcare_dataset.xlsx
- Dashboard: excel_dashboard.xlsx
- Presentation: project_presentation.pptx

## Outcome
This project helps hospitals identify operational patterns, optimize doctor workload, and track patient trends for better decision making.

## 🧠 Business Impact
The analysis helps improve service levels, reduce stock-outs, and enhance supply chain efficiency.
